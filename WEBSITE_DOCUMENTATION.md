## üéÆ ArcadiaX - The Flagship Product

### What is ArcadiaX?

ArcadiaX is a complete mixed reality social gaming platform that features drift karting and multiple party games. It's designed to transform empty warehouse spaces (400m¬≤+) into fully equipped MR gaming centers.

### Key Features:

**1. Mixed Reality Drift Karting**
- Physical drift karts integrated with Meta Quest 3 headsets
- Players see real karts, the arena, and other players
- Virtual game elements (obstacles, power-ups, paint splatters) overlaid in real-time
- 4-6 players per session

**2. Multi-Game Library**
The platform includes 5+ unique party games:

- **MR Drift Karting**: High-speed competitive racing with virtual obstacles
- **Paint Battle**: Color warfare where players paint the arena and opponents
- **Ghost Hunt**: Players hunt virtual ghosts while driving
- **Whack-a-Mole**: Hit virtual moles that pop up around the arena
- **Holy Cows**: Collect virtual cows while avoiding obstacles
- New games added regularly

**3. Complete Technology System**
- **Meta Quest 3 Headsets**: State-of-the-art MR devices
- **Custom Drift Karts**: Specially designed with integrated MR systems
- **Operator Management App**: Tablet/PC interface for managing games, scores, teams, playlists
- **Spectator & Interactive App**: "Meekijk & meespeel" (watch & play along) - spectators can view the action and interact with gameplay
- **Complete Infrastructure**: Server hosting, Wi-Fi routing, VR license management
- **Safety & Control**: Real-time monitoring, speed override, emergency stop systems

### How It Works:

**Step 1: Players Gear Up**
- Each player receives a Meta Quest 3 headset
- Gets seated in a custom drift kart with integrated MR controls

**Step 2: Arena Anchoring**
- System maps the physical arena (15√ó15m to 20√ó30m)
- Anchors virtual game elements precisely in real space

**Step 3: Game Selection**
- Operators choose from 5+ party games via management app
- Players see game overlay in their headsets

**Step 4: Mixed Reality Play**
- Players physically drive karts while interacting with virtual objects
- Real-time multiplayer competition

**Step 5: Spectator Engagement**
- Non-players watch action on screens
- Can interact through spectator app

**Step 6: Share & Replay**
- Game footage captured and stored
- Available for sharing, marketing, player re-engagement

### Venue Requirements:

**Minimum Space:** 15√ó15m (compact arena for 4 players)
**Optimal Space:** 20√ó30m (full arena for 6 players)
**Player Capacity:** 4-6 players per gaming session
**Pricing:** ‚Ç¨30 per player per session

## üìß Contact Form Flow

### User Journey:
1. User fills out form (name, email, message)
2. Clicks "Send Message"
3. Button shows "Sending..." with disabled state
4. Form submits to `/api/contact`
5. Backend processes request
6. SendGrid sends email to ggeesestudio@gmail.com
7. User sees success toast
8. Form resets automatically

### Email Format:
- **Subject:** "New Contact Form Submission from [Name]"
- **To:** ggeesestudio@gmail.com
- **Reply-To:** User's email (for easy response)
- **Content:** Styled HTML email with user's name, email, and message
- **Branding:** Purple accent colors matching website
